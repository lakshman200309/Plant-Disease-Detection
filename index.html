Welcome to Plant Disease Detection

This application helps you detect plant diseases by uploading an image of the plant. Navigate to the Upload Image section to upload an image and get a prediction.

About This Project

The Plant Disease Detection project aims to assist farmers and gardeners in identifying diseases in their plants through image recognition. By leveraging the power of deep learning and the InceptionV3 model, this application can accurately predict the type of disease affecting a plant based on an uploaded image.

Key Features:

  • Easy-to-use interface for uploading plant images.
  • Accurate disease prediction using a trained deep learning model.
  • Support for multiple types of plant diseases.

This project was developed to provide a quick and reliable tool for plant disease detection, helping to improve crop health and yield.
